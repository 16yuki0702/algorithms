# graph.go
  
  封装了邻接矩阵和邻接表表示，以及两种表示的互相转换
  
# bfs.go
  
  返回一个广度优先前趋子图， 前趋子图是邻接表还是邻接矩阵依赖于输入的图是邻接表还是邻接矩阵
  
# dfs.go
  返回深度优先前森林，反向边图，正向边图和交叉边图， 用栈替换了递归。用自定义linkedMap 代替map, 以解决key的顺序随机的问题。
  
  树边：
    parent -> vertex
    
  反向边：
    入栈时，如果为灰色，为反向边
    
  交叉边：
    入栈时，如果为黑色，为交叉边
    
  正向边：
    入栈时，如果parent已经存在，为正向边
